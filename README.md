# MOOC

Massive Open Online Course (MOOC) adalah metode pembelajaran daring (online) yang dirancang untuk partisipasi massal dan terbuka untuk siapa saja melalui internet. MOOC menawarkan berbagai mata pelajaran, seringkali gratis, dan memungkinkan fleksibilitas belajar mandiri tanpa batasan waktu, tempat, atau latar belakang pendidikan.

## Karakteristik Utama MOOC

- **Massive (Masif)**: Tidak ada batasan jumlah peserta, bisa diikuti ribuan orang sekaligus.
- **Open (Terbuka)**: Terbuka untuk umum, seringkali tanpa kriteria pendaftaran yang ketat, dan banyak yang gratis.
- **Online**: Seluruh materi, video, dan tugas diakses melalui web.
- **Course (Kursus)**: Terstruktur seperti perkuliahan konvensional, terkadang menyediakan sertifikat.

## Contoh Penyedia MOOC Populer

Contoh penyedia MOOC populer antara lain Coursera, edX, Udacity, dan platform kampus-kampus di Indonesia. Di Indonesia, MOOC juga digunakan untuk pelatihan dasar ASN/PPPK.
